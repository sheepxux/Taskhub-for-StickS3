TaskHub Web Bridge

Port is fixed to 5577 — the extension only has permission to reach the Host on that port. Change Host only between 127.0.0.1 and localhost.

The token must match the Host's token at ~/Library/Application Support/StickS3TaskHub/token. This extension reads only the open Gemini / Lovable / Perplexity tab title and sends it to your local Host — nothing leaves your machine.
